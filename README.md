# 🌿 GreenRupee — MSME ESG Scorer

> "Today, sustainability exists in India. Proof does not. We're building that proof."

## 🚀 Live Demo
🔗 https://Sashmitha-sree.github.io/greenrupee

## 📌 What is GreenRupee?
India's first ESG scoring platform built specifically for small businesses (MSMEs).
Answer 13 simple questions → Get your ESG score → Download a certificate →
Show it to banks and buyers.

## 😣 The Problem
63 million MSMEs in India pay fair wages, manage waste, and source sustainably —
but have NO document to prove it.

Result:
- Banks charge higher interest rates
- Export buyers cancel orders
- They lose money doing the right things

## ✅ Our Solution — 4 Features

| Feature | What it does |
|---|---|
| 🏭 MSME ESG Scorer | 20 questions → Score → PDF Certificate |
| 📊 ESG Fund Screener | E/S/G breakdown of every Indian ESG fund |
| 🌍 Live Impact Dashboard | See what your SIP did to the planet |
| 📋 BRSR Auto-Generator | Upload data → SEBI-ready report in 3 days |

## 🛠 Tech Stack

| Layer | Tool |
|---|---|
| Frontend | React.js + Vite |
| Backend | Python + FastAPI |
| Database | Supabase (PostgreSQL) |
| ESG Scoring | Rule-based Python engine |
| PDF Generation | ReportLab |
| Hosting | Vercel + Railway |
